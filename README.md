# QRCode

一個以純前端實作的 QR Code 工具箱，提供兩個主要模式：

- 文字轉 QR Code
- QR Code / 條碼解析

適合用來快速產生分享用 QR Code，或直接從圖片、剪貼簿與鏡頭讀取 QR Code / 條碼內容。

## 線上使用

直接開啟以下連結即可使用：

https://joengan.github.io/QRCode/

## 功能特色

- 產生 QR Code：輸入文字、網址或任意資料後，即可產生 SVG QR Code。
- 快速鍵支援：在輸入框內按 Ctrl + Enter（macOS 為 Cmd + Enter）可直接產生。
- 圖片解析：可選擇本機圖片檔解析 QR Code 或條碼。
- 拖曳解析：可直接將圖片拖曳進頁面中的解析區域。
- 剪貼簿解析：在解析頁籤中可直接貼上圖片後辨識。
- 鏡頭掃描：可手動啟動鏡頭進行即時掃描。
- 多鏡頭切換：若裝置存在多個鏡頭，可從下拉選單切換。
- 掃描輔助操作：支援滑鼠滾輪縮放、觸控縮放，且在部分裝置上可點擊對焦。
- 結果複製：成功解析後可一鍵複製辨識內容。

## 使用方式

### 1. 產生 QR Code

1. 開啟首頁後停留在「產生 QR Code」頁籤。
2. 在輸入框貼上文字、網址或其他內容。
3. 按下「產生 QR Code」按鈕，或使用 Ctrl + Enter / Cmd + Enter。
4. 需要重設時可按「清除內容」。

### 2. 解析圖片中的 QR Code / 條碼

切換到「解析 QR / 條碼」頁籤後，可以使用以下任一方式：

- 按「選擇圖片」後挑選圖片檔。
- 直接把 PNG、JPG、WEBP 等圖片拖進拖放區。
- 在該頁籤直接貼上剪貼簿中的圖片。

解析成功後，頁面會顯示：

- 解析內容
- 來源類型
- 條碼格式

並可透過「複製結果」按鈕將內容寫入剪貼簿。

### 3. 使用鏡頭掃描

1. 切換到「解析 QR / 條碼」頁籤。
2. 點選「使用鏡頭」。
3. 瀏覽器跳出權限要求時，允許網站使用攝影機。
4. 將 QR Code 或條碼對準畫面中央。
5. 掃描成功後，系統會自動停止鏡頭並顯示結果。

若裝置支援，還可使用以下操作：

- 滑鼠滾輪縮放預覽
- 觸控手勢縮放
- 點擊畫面對焦

## 本機執行

此專案為靜態網頁，無需建置流程，也不依賴後端服務。

### 直接開啟

可直接用瀏覽器開啟 index.html。

### 建議方式

若要使用鏡頭掃描，建議透過本機靜態伺服器啟動，例如：

- VS Code Live Server
- Python 簡易伺服器：python -m http.server 8000
- Node.js 靜態伺服器：npx serve
- Windows IIS

接著以 localhost 或 HTTPS 網址開啟頁面，避免瀏覽器因安全性限制而拒絕攝影機權限。

## 專案結構

- index.html：頁面結構與 UI 區塊。
- index.css：頁面樣式與互動狀態設計。
- index.js：QR 產生、圖片解析、鏡頭掃描與事件處理。
- html5-qrcode/2.3.8/html5-qrcode.min.js：掃描與解碼使用的前端函式庫。
- uqr@0.1.3/es2022/uqr.js：產生 QR Code SVG 使用的函式庫。

## 技術說明

- 使用原生 HTML、CSS、JavaScript 撰寫。
- QR Code 產生採用 uqr。
- 圖片與鏡頭解析採用 html5-qrcode。
- 鏡頭模式整合了鏡頭切換、縮放與部分裝置的對焦能力。

## 瀏覽器需求

建議使用支援現代 Web API 的桌面或行動裝置瀏覽器，例如：

- Microsoft Edge
- Google Chrome
- Safari
- Firefox

其中鏡頭權限、縮放與點擊對焦功能，仍會依不同裝置與瀏覽器支援程度而有所差異。
